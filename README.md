# Getting Started Guide

## Clone repository

```
git@github.com:alexvegasdev/city-tours.git
```

## Install dependencies

```
composer install

npm install
```

## Execute migrations

```
php artisan migrate
```

## Start servers

```
php artisan serve

npm run dev
```
